# 💬 Real-Time Chat App (MERN Stack + Socket.IO)

This is a full-stack real-time chat application built using the **MERN stack** with **Socket.IO** for instant messaging. Users can register, log in, and chat in real-time with others. It includes features like user online indicators, message history, and secure authentication.

---

## 🚀 Features

- 🔐 User Authentication with JWT (Register/Login)
- 🧑‍🤝‍🧑 One-to-one real-time messaging
- 📡 Real-time communication using **Socket.IO**
- 💬 Instant message updates without page refresh
- 📜 Chat history saved in MongoDB
- 👤 User list
- 🔒 Protected chat routes (auth middleware)
- 📱 Responsive design for mobile and desktop

---

## 🛠️ Tech Stack

| Area       | Technology                          |
|------------|-------------------------------------|
| Frontend   | React.js, Context API, CSS/Tailwind |
| Backend    | Node.js, Express.js                 |
| Database   | MongoDB with Mongoose               |
| Realtime   | Socket.IO                           |
| Auth       | JWT (JSON Web Tokens)               |
| Deployment |  Render                             |
